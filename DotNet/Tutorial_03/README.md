**Fuente:** OpenWebinars

## Descripción

Este curso de .NET Framework permite aprender a desarrollar aplicaciones desde cero utilizando el ecosistema completo de Microsoft. Está orientado a comprender la arquitectura por capas, la conexión con bases de datos SQL Server y el uso de Entity Framework para realizar operaciones CRUD.

> **Nota:** Todos los ejercicios y materiales pertenecen al autor original. Este repositorio se utiliza únicamente para fines de aprendizaje personal y documentación de los tutoriales.

---

## Habilidades que obtendrás

- Administrar y modelar una base de datos con SQL Server.
- Crear una arquitectura por capas (Presentación – Negocio – Datos) en Visual Studio.
- Diseñar y administrar una interfaz de usuario con WinForms.
- Desarrollar operaciones CRUD con Entity Framework sobre SQL Server.
- Compilar, depurar y publicar aplicaciones con Visual Studio.
