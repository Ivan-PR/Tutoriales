## Descripción

Aprende C# desde cero para sentar las bases de programación con Visual Studio y empieza a crear aplicaciones para .NET.

> **Nota:** Todos los ejercicios y materiales pertenecen al autor original. Este repositorio se utiliza únicamente para fines de aprendizaje personal y documentación de los tutoriales.

---

## Habilidades que obtendrás

- Dominar el lenguaje de C# desde 0  
- Usar Visual Studio  
- Comprender conceptos básicos de programación y cómo se aplican en C#  
- Entender conceptos de OOP y cómo se aplican en C#

---

## Contenido del tutorial

1. Introducción a C# y Visual Studio  
2. Tipos de datos y variables  
3. Operadores y estructuras de control  
4. Funciones y métodos  
5. Programación orientada a objetos (OOP)  
6. Ejercicios prácticos
