## Descripción

En este segundo curso de Symfony 5 podrás continuar ampliando tus conocimientos en los conceptos fundamentales del framework Symfony, permitiéndote crear una aplicación web completa al acabar la formación.

> **Nota:** Todos los ejercicios y materiales pertenecen al autor original. Este repositorio se utiliza únicamente para fines de aprendizaje personal y documentación de los tutoriales.

---

## Habilidades que obtendrás

- Aprenderás a fondo sobre Formularios
- Conocerás Doctrine
- Manejarás Ajax
- Aprenderás todo sobre eventos
- Sabrás gestionar usuarios
